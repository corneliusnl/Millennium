Master Boot Record

Hard disks can be divided into serveral partitions. One of the partitions is marked as active and is used for booting.

The information about how the disk is divided is located in the boot sector of the hard disk (sector 1, head 0, cylinder 0) and has a special format -- and is known as the master boot record.

Offset Size Description
0 1BEh Code which detects the active partition and boots the operating system from it.
1BEh 10h Partition entry 1
1CEh 10h Parition entry 2
1DEh 10h Partition entry 3
1EEh 10h Partition entry 4
1FEh 2 Partition signature (AA55h)
