Drive Naming Conventions

Hard Disks

hd[number][paritition]

number		A number designating the hard disk,
		with 0 referring to the first disk.
partition	A letter designating a partition on
		the hard disk, with 'a' referring to
		the first partition.  If not specified,
		it starts at the first sector.

Floppy Disks

fd[number]

Cdroms

cd[number]
