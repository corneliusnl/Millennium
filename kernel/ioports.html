Millennium I/O Ports

070-7F		NMI - Non-Maskable Interrupt and CMOS RAM
080-9F		DMA Page Register (74LS612)
0A0-0BF		Slave Interrupt Controller (8259A)
0C0-0DF		Slave DMA Controller (8237A)
0F0-0FF		80287 Math Coprocessor
1F0-1F8		Hard Disk
200-20F		Game Port Joystick Controller
258-25F		Intel Above Board
278-27F		Parallel Printer Port 2
2E8-2EF		Serial Port 4
2F8-2FF		Serial Port 2
300-31F		Prototype card
378-37F		Parallel Printer Port 1
380-38F		SDLC or Bisynchronous Communications Port 2
3A0-3AF		Bisynchronous Communications Port 1
3B0-3BF		MDA - Monochrome Adapter
3BC-3BE		Parallel Printer on Monochrome Adapter
3C0-3CF		EGA - Reserved
3D0-3D7		CGA - Color Graphics Adapter
3E8-3EF		Serial Port 3
3F0-3F7		Floppy Diskette Controller
3F8-3FF		Serial Port 1
