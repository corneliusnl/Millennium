FAT Filesystem

Boot Sector

The first sector of all FAT filesystems contain a 512 byte boot sector program which loads the necessary operating system files into memory and then executes them.

File Allocation Table

Next there comes a File Allocation Table which is often followed by another File Allocation Table in case the first one should become corrupt.
