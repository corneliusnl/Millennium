Millennium Driver Consortium
The purpose of the Millennium Driver Consortium is to provide a standard inferface for both hardware and software components. With these standards, hardware manufactors and even software developers (i.e. filesystem developers) will be able to develop a single driver that could be compiled and "plugged in" to any operating system. For operating system developers, this would mean having the latest versions of the drivers compiled for their operating system available for download at their online site. For hardware manuafactures, this would mean creating one driver that will run on any operating system complying with the standards set by the Millennum Driver Consortium. But most importantly, for the end user, it would mean operating system independance!
